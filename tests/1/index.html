Test Application

This is a test application for the Johnny's web view project.

Johnny's web view is an application written in C++ that loads a web page from the Internet or from the local machine.

Features

The HTML applications for desktop are very easy to develop. The project contains the following features:

  • load any HTML page from Internet or from the local machine
  • simple API
  • cross platform
  • contributing is simple
  • open source project

API

Still in developement. The following feature will be or were already built:

  • set window width (✔)
  • set window height (✔)
  • load HTML pages (✔)
  • close window
  • set window title
